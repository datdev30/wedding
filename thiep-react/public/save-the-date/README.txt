Save the date — ảnh section 3
=============================

Đặt ảnh vào thư mục này và khai báo trong manifest.json, ví dụ 2 ảnh:

  "images": ["anh-ben-trai.jpg", "anh-ben-phai.jpg"]

Thứ tự mặc định (2 ảnh — theo cột trái / phải):
  • Ảnh 1 → cả cột trái (khung lớn IMAGE6 + khung nhỏ IMAGE7)
  • Ảnh 2 → cả cột phải (khung lớn IMAGE8 + khung nhỏ IMAGE9)

Nếu khai báo đủ 4 ảnh: gán lần lượt IMAGE6 → IMAGE7 → IMAGE8 → IMAGE9.

Sau khi sửa, F5 trang.
